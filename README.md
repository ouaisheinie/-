# 钟小庆耳鸣症状表现
## 耳鸣定义

耳鸣是累及听觉系统的许多疾病不同病理变化的结果，病因复杂，机制不清，主要表现为无相应的外界声源或电刺激，而主观上在耳内或颅内有声音感觉。在临床上它既是许多疾病的伴发症状，也是一些严重疾病的首发症状（如听神经瘤）。  

以上摘自百度百科

## 症状

呈多样性，单或双侧，或头鸣，持续或间歇，晚上睡觉前或安静环境中感觉明显。
